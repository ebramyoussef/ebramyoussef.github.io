Nonlinear waveguide SSFM numerical simulation

This MATLAB numerical simulation utilizes the Split-Step Fourier Transform (SSFM) algorithm to numerically solve the Nonlinear Schr√∂dinger equation (NLSE) for a TE polarized Gaussian beam propagating through a nonlinear waveguide geometry. The specific geometry of this simulation was a thin layer of a nonlinear material (ITO; Indium-Tin-Oxide), with a glass substrate to one side and air to the other. The Gaussian beam is typically assumed to start propagating on the air side of this geometry, and so the nonlinear interactions at the air-ITO interface are the ones studied. This simulation ignores the out-of-plane dimension (treating the problem using a 2D scalar formalism) and assumes saturable Kerr nonlinearity with and without linear and nonlinear losses. The beam parameters (i.e., spot size, wavelength, grazing angle, etc.) are changed as to find the optimal parameters for a variety of nonlinear effects. This simulation was developed for a project I work on with Saleem Iqbal (PhD candidate at Boyd group) aimed at experimentally exciting and observing surface spatial solitons at the interface of air-ITO in a similar geometry, and so the main purpose of this simulation is to find the optimal parameters for soliton excitation as theoretically proposed by W. J. Tomlinson in the 1980s. However, this simulation was used to observe a variety of other exciting nonlinear effects. This project is still ongoing.
